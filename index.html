About Crypto Report

Dataset

The report uses a dataset generated from a web scraper built in R.

Jupyter Notebook

Jupyter Notebooks was used to create the report. It allows switching between R, Markdown, and Python. The majority of the report was made using Python.

Regression Models

The report uses 5 different Regression Models to help predict prices. These are Lasso, Ridge, Elastic Net, Random Forest, and Gradient Boosted.

Issues & Solutions

Bitcoinaverage API

I couldn't get the Bitcoinaverage API to return all the data I required for the report.

R scraper

Instead of the Bitcoinaverage API, I used an R scraper to gather all the data I needed.

Flask App

Migrating the Notebook into a Flask App was producing lots of errors and essentially required starting anew.

Convert Notebook to HTML

Instead, I converted the Notebook to HTML and integrated it into this website.
